Existen tres operadores Booleanos, que se usan para comparar valores Booleanos, que son and(&&), or(||), xor(xor) y not(!).

AND o && (Y)

Expreción Evalua como

True && True True
### OR o
Expreción Evalua como
- -
False \|\| False False

XOR (O exclusiva)

Expreción Evalua como

False XOR True True

NOT (NO)

Expreción Evalua como
